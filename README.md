# node-react-user-sample
> Node.js and React.js User Sample.

<br>

## Development Environment
- IDE : Visual Studio Code
- Language : HTML5, CSS3, JavaScript (ES6)
- Framework : Node.js (v18.14.2)
- Library : React.js (v18.2.0)
- DBMS : MongoDB (v4.0)

<br>

## Main Function
- Home : /
- User
  - Login : /login
  - Logout : /logout
  - Join : /join

<br>

## Install & Run Node.js
```shell
npm install
npm run dev
```
